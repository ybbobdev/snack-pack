# Snack Pack
Boiler plate collection of applications needed for a full stack web site.
 - Typescript
 - Tailwind
 - Nextjs
 - Sequilize (TBD)
 - Postgres (TBD)
 - GCP configs (TBD)

See "Duplicating a repository" to start a new project without forking

Local development
```
nvm use
yarn install
yarn dev
```
